# 🚀 AARU CLI - Quick Installation Guide

## 🐧 Linux (One Command)

```bash
git clone https://github.com/aarushlohit/GIT_PROTOCOL.git && cd GIT_PROTOCOL && ./install.sh
```

---

## 🪟 Windows (One Command)

```batch
git clone https://github.com/aarushlohit/GIT_PROTOCOL.git && cd GIT_PROTOCOL && install_windows.bat
```

---

## 🔥 Build Standalone Executable

### Linux:
```bash
./build_exe.sh
sudo cp dist/aaru /usr/local/bin/
```

### Windows:
```batch
build_exe.bat
REM Then add dist folder to PATH or copy dist\aaru.exe to C:\Windows\System32
```

---

## ⚡ Quick Start

```bash
aaru aaru       # Show banner
aaru init       # Initialize repo
aaru save "msg" # Quick commit
aaru sync       # Push/pull
aaru --help     # Show all commands
```
